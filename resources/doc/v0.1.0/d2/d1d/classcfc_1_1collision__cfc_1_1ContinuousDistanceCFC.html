Detailed Description

template<typename G1, typename G2>
class cfc::collision_cfc::ContinuousDistanceCFC< G1, G2 >

Narrow phase continuous proximity queries between two 3D convex bodies with smooth boundaries. Boundary parameterized by outward gradient/normal. Optimization methods to be implemented by subclasses.
