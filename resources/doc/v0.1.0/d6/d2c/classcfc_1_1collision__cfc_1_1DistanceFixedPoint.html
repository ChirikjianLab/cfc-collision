Detailed Description

template<typename G1, typename G2>
class cfc::collision_cfc::DistanceFixedPoint< G1, G2 >

Narrow phase proximity queries between two convex bodies. Bodies are parameterized by outward gradient (un-normalized). Collision detection is solved by fixed-point iteration method.
